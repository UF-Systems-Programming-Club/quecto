This isn't a super strict grammar with like BNF or anything, just a nice reference for the future when changing the parser.

'*' means 0 or more times
'+' means 1 or more times

program = *(statement) EOF

statement = expr '\n' or
            declaration '\n' or
            return '\n'

identifier = (_|a-z|A-Z)(a-z|A-Z|0-9|_)*
declaration = identifer : {type} = expr
return = return expr
expr = ( '(' expr ')' or number ) *(operator expr)
